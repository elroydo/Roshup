# Roshup

Roshup is a simple, browser-based RotaCloud shift uploader.

Live URL: https://elroydo.github.io/Roshup/

It helps you:

- connect to the RotaCloud API
- load and parse a weekly workbook
- validate shifts before upload
- dry-run duplicate checks
- upload valid shifts in controlled batches

## Quick Start

1. Add your API token (or load from `.txt`)
2. Connect API
3. Select workbook, sheet, year, and week
4. Parse sheet
5. Set location, role, and upload options
6. Run **Check Duplicates** (recommended)
7. Click **Upload Valid Shifts**

## Highlights

- Guided 3-step workflow
- Live validation board and preview
- Execution console and status updates
- Light/dark mode toggle
- Optional local token remember setting
